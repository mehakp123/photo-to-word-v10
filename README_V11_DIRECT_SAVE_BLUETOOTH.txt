Photo to Word v11 - Direct Save + Bluetooth

What changed from v10:
- Save Word file saves a real .docx file to Downloads/Photo to Word.
- Save pictures to folder saves real .jpg files to Pictures/<folder name>.
- Bluetooth Word file shares only the Word file.
- Bluetooth pictures shares actual image files, not a ZIP.
- Bluetooth Word + pictures ZIP uses ZIP only for sharing both together.
- File extensions are preserved, so files should not appear as unknown files.

Build:
Upload these files to a GitHub repo and run the included GitHub Actions workflow.
