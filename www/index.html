Photo to Word

v11 Direct Save + Bluetooth. Save Word as .docx directly and save photos directly into a phone Pictures folder.

1) Upload photos

APP VERSION: v11 Direct Save + Bluetooth. Saves real .docx and .jpg files directly; Bluetooth buttons only share.

Tip: select all required photos together. New photos will be resized automatically using the settings below.

Word picture layout: every photo is inserted as a separate floating picture with Square wrapping. In Microsoft Word you can select any photo, drag it anywhere on the page, resize it, rotate it, or change its wrap/layout.
Direct save: Save Word creates a real .docx file in Downloads/Photo to Word. Save pictures creates real .jpg files in a folder under Pictures, using the folder name you type.
Bluetooth: Share buttons only open the Android share screen for Bluetooth. Android still requires you to select Bluetooth/device manually. ZIP is used only for sharing Word + pictures together.
