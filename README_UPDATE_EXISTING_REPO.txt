PHOTO TO WORD v10 - NATIVE SAVE + BLUETOOTH

This package must replace your existing GitHub repository files.
The new app screen will show:
APP VERSION: v10 Native Save + Bluetooth
and buttons:
- Save / Share Word file
- Save / Share pictures ZIP
- Save / Share Word + pictures ZIP
- Bluetooth Word file
- Bluetooth pictures ZIP
- Bluetooth Word + pictures ZIP

IMPORTANT:
After downloading the new APK, uninstall the old Photo to Word app first.
Debug APKs built on GitHub may use a different debug signature each run, so Android may keep the old app if you do not uninstall it.

Codespaces update commands:
unzip -o photo_to_word_android_apk_repo_v10_NATIVE_SAVE_BLUETOOTH.zip
git add .
git commit -m "Add v10 native save and Bluetooth buttons"
git push

Then go to Actions and run Build Android APK.
Download artifact named:
Photo-to-Word-v10-Native-Save-Bluetooth-APK
